# WI Film Tax Credits

Landing page for Wisconsin's new (2026) Film Production Tax Credit consulting service.
Primary goal: lead capture / contact.

Built with the same lightweight framework as `iltaxcredits-website` — Vite + plain HTML + a single `styles.css`.

## Develop
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
Output is in `dist/`.

## Program facts (as of April 2026)
- 30% tax credit on qualified WI production expenses (refundable / transferable / non-refundable components)
- $1M cap per project
- $5M statewide annual cap
- Min spend: $100k for 30+ minute productions, $50k for shorter works
- Retroactive to expenditures on/after Jan 1, 2026
- Administered by Film Wisconsin (WI Dept of Tourism)
- Launched Feb 17, 2026 by Gov. Evers
